================================================================================
CONSOLE COMMANDS (Available Globally)
================================================================================

getEventSummary()   - Show statistics summary
getAllEvents()      - Get all captured event data
exportEvents()      - Export as JSON format
clearEvents()       - Clear all captured data

Example Usage:
--------------
1. Open console (F12)
2. Type: getEventSummary()
3. Press Enter
4. See your event statistics!

================================================================================
TROUBLESHOOTING
================================================================================

Problem: No events showing
Solution: 
- Check console for errors
- Verify script is loaded
- Try: console.log(window.eventTracker)

Problem: Too many events
Solution:
- Normal! Mouse/scroll create many events
- Use clearEvents() to reset
- Filter by event type if needed

Problem: Console is cluttered
Solution:
- Click "Clear console" button
- Use console filters
- Close/reopen Developer Tools
